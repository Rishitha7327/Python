# User Profile Validation System (Day-1)

This Python program checks whether the entered **Full Name, Email ID, Mobile Number, and Age** are valid or not using basic conditions.

## Validation Rules
- **Full Name:** should not start/end with space and must contain at least one space  
- **Email:** must contain `@` and `.` and should not start with `@`  
- **Mobile Number:** must be 10 digits, only numbers, and should not start with `0`  
- **Age:** must be between 18 and 60  

## Output
Prints:
- `User profile is VALID` ✅  
or  
- `User profile is INVALID` ❌
